Onboarding

Go from empty box to usable agent workspace.

Set identity, connect docs, review policy, seed memory, then activate the always-on agent loop.

02Ingest documents

Bring in SharePoint, Drive, CRM exports, and restricted folders. PII is tagged before retrieval.

03Build context graph

04Turn agents on
